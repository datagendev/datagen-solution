# DataGen Solution

Knowledge base and Claude Code skills for [DataGen](https://datagen.dev) -- the deployment platform for Claude Code agents.

## What's Inside

- **`datagen.md`** -- Product overview: what DataGen is, how it works, pricing, and getting started
- **`tools/`** -- Reference docs for 150+ MCP tools available on the platform (Slack, Gmail, Notion, Linear, Heyreach, Instantly, Firecrawl, and more)
- **`.claude/skills/`** -- Claude Code skills that use DataGen context, including the LinkedIn Agent Recommender

## What Is DataGen

DataGen deploys Claude Code agents. You build locally, push to GitHub, and DataGen handles the rest:

- **One-click deploy** to webhook URL or cron schedule
- **MCP tool gateway** -- tools work the same locally and in production
- **Slack and email channels** -- agents communicate through built-in messaging
- **Human-as-a-tool** -- agents run autonomously and only reach humans for judgment calls

## Quick Start

1. Build your agent in Claude Code
2. Push to GitHub
3. Install the DataGen GitHub App
4. Deploy with one click

See [datagen.dev](https://datagen.dev) for full docs.
